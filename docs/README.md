## Getting Started

### Installation

1. Visit the Chrome Web Store
2. Search for "Omni - AI Conversation Organizer"
3. Click "Add to Chrome"
4. Confirm the installation

### First Launch

When you first open Omni:

1. You'll see the welcome screen with a quick tour
2. Optionally, sign in with Google or GitHub for cloud sync
3. Create your first project to organize conversations

### Quick Setup

**Step 1: Create a Project**
- Click the "+" button in the sidebar
- Name your project (e.g., "Work Projects", "Learning")
- Choose an icon and color

**Step 2: Enable Auto-Capture**
- Visit ChatGPT, Claude, Gemini, or Grok
- Omni automatically detects and captures conversations
- No manual action required

**Step 3: Explore Your Data**
- Open Omni to see captured conversations
- Use search to find specific content
- Switch between AI models using the transfer feature

## Core Concepts

### Projects

Projects are containers for organizing your AI conversations:

- **Create projects** for different workflows (e.g., "Coding", "Research", "Writing")
- **Add conversations** to projects manually or automatically
- **Pin important items** to the top of your project
- **Archive completed** projects to keep things tidy

### Conversations

A conversation is a single chat session with an AI model:

- **Captured automatically** from supported platforms
- **Contains the full history** of messages and responses
- **Can be exported** in Markdown, JSON, or PDF formats
- **Searchable** across all your conversations

### AI Switching

Omni's signature feature - transfer context between AI models:

1. Open a conversation in Omni
2. Click "Switch AI" in the toolbar
3. Select the target AI model
4. Omni reformats the context for the new model
5. Click "Open" to continue in the new platform

**How it works:**
- Omni extracts the key context from your conversation
- Reformats prompts for the target AI's style
- Opens the target platform with a pre-formatted prompt
- You can continue the conversation seamlessly

### Notes

Quick-capture notes alongside your conversations:

- Create notes within any project
- Format with rich text (headers, lists, links)
- Link notes to specific conversations
- Search notes alongside conversations

### Files

Upload and organize files with your workspace:

- Drag and drop files into projects
- Preview common formats (PDF, images, text)
- Link files to conversations for context
- Sync to cloud with Pro subscription

## Features

### Automatic Capture

Omni automatically captures your conversations when you use:
- ChatGPT (chat.openai.com)
- Claude (claude.ai)
- Gemini (gemini.google.com)
- Grok (x.ai / grok.x.ai)

**What gets captured:**
- Full conversation history
- Session metadata (date, model used)
- Your project assignment (if set)

**What doesn't get captured:**
- Sensitive personal data
- File uploads (just references)
- Payment information

### Universal Search

Search across everything in your workspace:

- **Conversations**: Full-text search through all messages
- **Notes**: Search note titles and content
- **Files**: Search file names and extracted text
- **Projects**: Filter by project, date, AI model

**Search Tips:**
- Use `#tag` to search by tag
- Use `@project` to search within a project
- Use `before:` and `after:` for date ranges
- Use `model:claude` or `model:chatgpt` to filter by AI

### Timeline View

See your AI interactions over time:

- Daily, weekly, monthly views
- Activity heatmaps
- Trend analysis
- Usage statistics

### Cloud Sync (Pro)

Sync your workspace across devices:

- Automatic background sync
- End-to-end encryption
- Conflict resolution
- Offline support

### Backups

Never lose your data:

- **Automatic backups** (Pro): Daily backups with 30-day retention
- **Manual backups**: Create backups on demand
- **Restore**: Restore specific items or complete workspace

### Export & Share

Export your conversations:

- **Markdown**: Clean, readable format
- **JSON**: Structured data for analysis
- **PDF**: Formatted for sharing
- **HTML**: Web-ready format

## Integrations

### GitHub

Connect your GitHub account:

1. Go to Settings > Integrations > GitHub
2. Click "Connect GitHub"
3. Authorize Omni
4. Select repositories to sync

**Features:**
- Link conversations to GitHub issues
- Create issues from conversations
- Sync code snippets
- Track AI-assisted development

### Google Drive

Connect Google Drive:

1. Go to Settings > Integrations > Google
2. Click "Connect Google Drive"
3. Authorize Omni
4. Select folders to sync

**Features:**
- Upload files from Drive
- Save exports to Drive
- Sync notes to Drive
- Share Omni content

### Notion

Export to Notion:

1. Go to Settings > Integrations > Notion
2. Click "Connect Notion"
3. Authorize Omni
4. Select destination pages

**Features:**
- Export conversations as Notion pages
- Sync notes to Notion
- Create databases from projects

## Privacy & Security

### Data Storage

**Local-First Design:**
- All data is stored locally on your device
- Cloud sync is optional
- You control what gets synced

**Encrypted Cloud Sync:**
- End-to-end encryption
- Keys never leave your device
- Even Omni can't read your synced data

### Data We Collect

**Automatically Collected:**
- Usage metrics (opt-in, anonymous)
- Error reports (opt-in, for debugging)
- Performance metrics (opt-in)

**Never Collected:**
- Conversation contents (unless cloud sync is on)
- Personal information
- Browsing history outside supported platforms

### Your Rights

- **Export**: Download all your data
- **Delete**: Permanently delete everything
- **Access**: Request a copy of stored data
- **Portability**: Transfer data to other services

### GDPR Compliance

- Located in the US
- Data processing agreements available
- Data deleted upon request
- No cross-border data transfers without consent

## Troubleshooting

### Conversations Not Capturing

1. Make sure you're on a supported platform
2. Check that Omni extension is enabled
3. Refresh the page
4. Check console for errors (F12)

### Sync Issues

1. Verify you're signed in
2. Check internet connection
3. Try manual sync from Settings
4. Clear cache and retry

### Performance Issues

1. Clear old conversations
2. Reduce sync frequency
3. Disable unused integrations
4. Restart browser

### Login Problems

1. Clear Omni extension storage
2. Re-authenticate
3. Check email for verification
4. Contact support

## FAQ

**Q: Is Omni free?**
A: Yes! Omni has a generous free tier with 3 projects and 100 conversations. Upgrade to Pro for cloud sync, AI switching, and more.

**Q: Does Omni read my conversations?**
A: No. Conversations are stored locally on your device. Cloud sync is encrypted end-to-end - even Omni can't read your data.

**Q: Which AI platforms are supported?**
A: Currently: ChatGPT, Claude, Gemini, and Grok. More platforms coming soon.

**Q: Can I use Omni offline?**
A: Core features work offline. Cloud sync requires internet connection.

**Q: How does AI switching work?**
A: Omni extracts context from your conversation and reformats it for the target AI model. It opens the target platform with your context ready.

**Q: Is my data secure?**
A: Yes. Local-first architecture means your data stays on your device. Cloud sync uses end-to-end encryption.

**Q: Can I export my data?**
A: Yes, export anytime in Markdown, JSON, PDF, or HTML formats.

**Q: How do I cancel my subscription?**
A: Cancel anytime from Settings > Subscription. Your access continues until end of billing period.
